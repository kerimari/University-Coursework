## 🛠️ Technical Highlights

* **Memory Management:** Extensive use of `malloc`, `free`, and pointer arithmetic to manage dynamic data structures.
* **Data Persistence:** Implementation of binary file CRUD operations with `fseek` and `fwrite`.
* **System Programming:** Direct interaction with Unix system calls (`open`, `read`, `lseek`) and Signal Handling.
* **Professional Standards:** Modular code design with comprehensive English documentation and structured logging.
